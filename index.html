JUO Legacy Scanner Usage Guide

Step 1: Start Camera and grant permission.

Step 2: Adjust Page Format, DPI, and Image Quality.

Step 3: Capture each page one at a time.

Step 4: Click Download PDF after all pages are captured.

Step 5: Use Image Quality slider to reduce file size for large PDFs.

Tips: Use a stable surface, good lighting, and medium DPI/quality for large documents (100+ pages).

Note: Only one page is kept in memory at a time to prevent crashes on mobile devices.
